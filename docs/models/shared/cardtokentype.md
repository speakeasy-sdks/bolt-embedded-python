# CardTokenType

Used to define which payment processor generated the token for this credit card.



## Values

| Name                      | Value                     |
| ------------------------- | ------------------------- |
| `VANTIV`                  | vantiv                    |
| `APPLEPAY`                | applepay                  |
| `BOLT`                    | bolt                      |
| `STRIPE`                  | stripe                    |
| `PLCC`                    | plcc                      |
| `APPLEPAY_ENCRYPTED_BLOB` | applepay_encrypted_blob   |
