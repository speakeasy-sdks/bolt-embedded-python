# TokenType

Used to define which payment processor generated the token for this credit card.  For those using Bolt's tokenizer, the value must be `bolt`.



## Values

| Name       | Value      |
| ---------- | ---------- |
| `VANTIV`   | vantiv     |
| `APPLEPAY` | applepay   |
| `BOLT`     | bolt       |
| `STRIPE`   | stripe     |
| `PLCC`     | plcc       |
